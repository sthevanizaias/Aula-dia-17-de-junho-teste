# Meus Flashcards de Programação

## Pergunta: O que é uma variável?
---
**Resposta:** É um espaço na memória do computador usado para armazenar dados que podem ser modificados durante a execução do programa.

###

## Pergunta: Qual a diferença entre `const` e `let` no JavaScript?
---
**Resposta:** `let` permite que você altere o valor da variável depois. `const` cria uma variável de leitura obrigatória cujo valor não pode ser reatribuído.

###

## Pergunta: O que significa HTML?
---
**Resposta:** HyperText Markup Language (Linguagem de Marcação de Hipertexto). Não é uma linguagem de programação, mas sim de marcação.
